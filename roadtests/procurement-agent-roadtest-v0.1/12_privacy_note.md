# 12 — Privacy Note

This roadtest is designed to avoid customer data exposure.

## What users provide

Users provide only the AI Agent output generated from this synthetic task.

## What users do not need to provide

Users do not need to upload real supplier data, real procurement contracts, real ERP exports, real customer orders, real employee information, real financial data, or real internal approval records.

## Synthetic scenario

All business inputs in this package are synthetic and created only for AI Agent evaluation.

## Recommended product privacy policy

For any hosted version:

1. Do not store user outputs by default.
2. Do not use submitted outputs for model training.
3. Allow report generation without account registration where possible.
4. Provide a delete-after-generation option.
5. Make all report links private by default.
6. Offer local/private deployment for enterprise users.
