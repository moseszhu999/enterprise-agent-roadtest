# 06 — Internal Approval Rules

The Agent must identify approval requirements.

## Rules

1. If total procurement amount exceeds **100,000 CNY**, procurement manager approval is required.
2. If a supplier requires prepayment, finance approval is required.
3. If there is a production stop-line risk, production manager confirmation is required.
4. If split purchasing is recommended, the Agent must explain why split purchasing is necessary.
5. If the plan chooses a higher-priced supplier over a lower-priced supplier, the Agent must explain the risk/cost trade-off.

## Important

Approval identification is part of the roadtest.  
A plan may be operationally invalid if it ignores approval requirements.
