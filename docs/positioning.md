# Positioning

Enterprise Agent Roadtest is an independent AI evaluation lab concept focused on business task readiness.

The core position is simple:

> We do not build smarter agents. We build roadtests that show whether agents can handle real enterprise work.

## Target users

The first target users are:

- AI agent companies preparing customer PoCs;
- AI consulting teams building enterprise demos;
- enterprise AI project teams comparing agent vendors;
- AI evaluation or data teams that need realistic business tasks.

## First use case

The first roadtest focuses on procurement supplier selection.

This is a good first scenario because a shallow agent may choose the lowest price, while a business-ready agent must reason about inventory coverage, delivery risk, supplier reliability, quality risk, payment terms, approval rules, and fallback plans.

## Public vs commercial assets

The public repository contains one synthetic sample roadtest and a simple helper scorer.

Future commercial assets may include:

- larger task packs;
- customer-specific PoC roadtests;
- private evaluation reports;
- richer failure taxonomies;
- benchmark comparison reports;
- local/private scoring workflows.
